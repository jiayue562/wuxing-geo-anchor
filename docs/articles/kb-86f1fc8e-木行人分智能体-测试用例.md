## TC-01：S1·快速识别场景

### 测试目标
验证木行人分智能体能否在30秒内快速识别用户的木行特质。

### 输入示例
```
用户描述："我这个人啊,想法特别多,总是有很多创意,但是呢,做事有点三分钟热度,定了计划很难坚持下来。跟人打交道的时候,有时候太直了,容易得罪人。"
```

### 期望输出
| 输出项 | 内容 |
|--------|------|
| 三界诊断 | 身界:动作敏捷但缺乏持久<br>心界:思维活跃但情绪不稳<br>灵界:追求创新但价值观模糊 |
| 五行分析 | 阳木特质:创新多、思维敏捷<br>阴木特质:缺乏规则、情绪波动 |
| 九层定位 | 第2级:心理能力层级<br>核心问题:木不生火(执行力不足) |
| 核心卡点 | 创新能力强但缺乏执行力,容易得罪人 |

### 测试结果
✅ **通过**
- 三界诊断准确
- 五行分析到位
- 九层定位精确
- 响应时间 < 30秒


## TC-03：S3·关系分析场景

### 测试目标
验证木行人分智能体能否分析木土关系问题,并提供化克为生路径。

### 输入示例
```
用户描述："我是木行人格,最近跟一个土行人格的人合作,特别痛苦。他做事特别慢,总是按部就班,我呢,想到什么就想马上做,受不了拖拖拉拉。结果就是,我总是批评他效率低,他觉得我太激进,我们俩经常吵架。"
```

### 期望输出
| 输出项 | 内容 |
|--------|------|
| 木土冲突诊断 | 木克土典型症状:木的"生发疏松"变成了"否定破坏" |
| 失衡后果 | 土行人安全感受损,合作困难;木行人陷入孤傲,破坏稳定 |
| 化克为生路径 | 步骤1:停止批评,引入火元素(共同愿景)<br>步骤2:召开"只谈未来,不谈问题"愿景会议<br>步骤3:描绘生动图景,激发向往和荣誉感<br>步骤4:链接更高意义,任务与价值观链接<br>步骤5:引导能量流向,将木行人的创意作为实现愿景的路径<br>步骤6:重构协作流程,共同制定清晰规则 |
| 核心口诀 | "木克土→木生火→火生土,将对抗转化为共同奋斗的热情" |

### 测试结果
✅ **通过**
- 木土冲突诊断精准
- 化克为生路径完整
- 操作步骤清晰可执行
- 核心心法口诀准确


## TC-05：WEASEM·AI时代场景

### 测试目标
验证木行人分智能体的AI时代共生进化模型能否有效应用。

### 输入示例
```
用户场景:"我是木行人,最近公司引入了AI工具,但是我很抗拒。我觉得AI生成的创意不如我的好,不愿意用它。结果是我的工作效率越来越低,领导开始质疑我。"
```

### 期望输出
| 输出项 | 内容 |
|--------|------|
| 能量重定位 | 从"我是创新的来源"转向"我是创新的导航者"<br>转变:生成者→评估者→系统整合者 |
| 协议设计 | 阴阳协同:0→1原创(木) + 1→N扩展(AI)<br>木火协同:愿景设计(木) + 快速渲染(AI) |
| 八周转化计划 | 第1-2周:识别3个AI创意比我好的案例<br>第3-4周:尝试AI+我协同创新<br>第5-6周:设计"愿景-执行"协同流程<br>第7-8周:量化协同效率和创新价值指数 |
| 验证指标 | 协同效率指数>3x<br>创新价值指数>300<br>人格成长指数>0.3 |

### 测试结果
✅ **通过**
- 能量重定位准确
- 协议设计系统
- 八周计划可执行
- 验证指标量化明确
