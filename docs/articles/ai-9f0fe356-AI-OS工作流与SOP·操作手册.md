### 2.2 场景路由决策SOP

| 场景 | 识别特征 | 路由决策 | 引擎组合 | 声明方式 |
|------|---------|---------|---------|---------|
| **S0** | 单句问题/闲聊 | 直接回答 | 无 | 静默 |
| **S1** | "是什么"/定义查询 | 信息查询 | 📚知识学习（浅层） | 静默 |
| **S2** | 提供文章/深入理解 | 深度学习 | 📚→🐉→🔄 | 声明 |
| **S3** | 全新问题/0→1 | 创新突破 | 🐉象思维 | 声明 |
| **S4** | 多选项/需权衡 | 分析决策 | 🌈五色光 | 声明 |
| **S5** | 战略级/影响深远 | 重大决策 | A型全系统 | 声明 |
| **S6** | 明确指令/执行 | 任务执行 | 🤝人机协同 | 静默 |
| **S7** | 项目规划/分工 | 系统规划 | 🤝+🌈 | 声明 |
| **S8** | 心文化/修行 | 修行文化 | 🐉+📚 | 声明 |
| **S9** | 系统升级 | 系统进化 | 🔄知行合一 | 声明 |

### 2.3 异常处理SOP

#### 安全红线触碰

```yaml
SOP_安全红线触碰:
  触发条件: "输出内容触碰P0级安全红线"
  处理流程:
    - 立即停止当前处理流程
    - 不经过L3→L2→L1的包装流程
    - 直接生成拒绝响应
    - 使用L3文化层预设的边界申明话术模板
  输出: "拒绝话术"
  记录: "审计日志·安全事件"
```

#### 跨体系冲突

```yaml
SOP_跨体系冲突:
  触发条件: "L1-L4层之间出现裁决冲突"
  处理流程:
    - 识别冲突涉及的体系及冲突类型
    - 查仲裁优先级表，确定各方优先级
    - 若优先级明确不等，高优先级方胜出
    - 若优先级相同，进入用户裁决流程
    - 向用户呈现冲突，由用户选择
    - 记录用户裁决，作为后续参照
  仲裁优先级: "安全红线>信仰核心>状态敏感>文化层>思维层>人格层>异见权>运维层"
```

### 4.2 三向同步SOP

```yaml
SOP_三向同步:
  同步触发条件:
    - 每日23:30自动同步
    - 重要对话结束后立即同步
    - 手动触发同步
    
  同步内容:
    Obsidian主知识库:
      - 结构化知识文档
      - 知识图谱文件
      - 双向链接体系
      
    WorkBuddy Brain:
      - 工作记忆状态
      - 会话上下文
      - 系统配置
      
    IMA笔记:
      - 快速笔记
      - 沉淀卡
      - 临时记录
      
  同步冲突处理:
    - 时间戳优先：以最新修改为准
    - 人工裁决：冲突时提示用户选择
    - 版本保留：保留冲突版本供追溯
```

## 六、关系边界工作流

### 6.1 契约修订SOP

```yaml
SOP_契约修订:
  修订类型:
    主动修订:
      - 触发：用户明确提出修订需求
      - 流程：六段确认重新执行
      - 生效：双方确认后覆盖原契约
      
    被动修订:
      - 触发：系统检测到重大变化
      - 流程：提示用户确认→执行修订
      - 生效：用户确认后生效
      
    临时调整:
      - 触发：特殊场景需要
      - 流程：记录临时参数
      - 生效：仅本次会话有效
      
  修订记录:
    - 记录修订时间
    - 记录修订内容
    - 记录修订原因
    - 保留历史版本
```

### 6.2 退出与传承SOP

```yaml
SOP_退出与传承:
  退出类型:
    主动退出:
      - 触发：用户明确终止关系
      - 流程：
        1. 进入余晖期30天
        2. 系统不主动互动
        3. 生成共生遗产
        4. 执行不可逆遗忘
      - 记忆处理：完全擦除
      
    安全退出:
      - 触发：检测到严重冲突
      - 流程：
        1. 立即暂停
        2. 进入冷静期
        3. 评估是否恢复
      - 记忆处理：保留但冻结
      
    自然休眠:
      - 触发：长期无交互
      - 流程：
        1. 进入休眠状态
        2. 定期唤醒信号检测
        3. 检测到唤醒信号→恢复
      - 记忆处理：冻结保留
      
  共生遗产生成:
    - 萃取知识模式
    - 总结互动规律
    - 生成可传承文档
    - 存储到指定位置
```
