## 五、五行分智能体调度规则

### 5.1 单智能体调用

| 场景 | 调用分智能体 | 场景示例 |
|------|-------------|---------|
| 只涉及木行 | 木行人分智能体 | "我是木行人，最近发怒比较多" |
| 只涉及火行 | 火行人分智能体 | "火行人如何转化嗔恨？" |
| 只涉及土行 | 土行人分智能体 | "土行人总是抱怨怎么办？" |
| 只涉及金行 | 金行人分智能体 | "金行人如何不过度追求完美？" |
| 只涉及水行 | 水行人分智能体 | "水行人焦虑怎么转化？" |

### 5.2 双智能体组合

| 组合 | 调用分智能体 | 场景示例 |
|------|-------------|---------|
| 木+火 | 木行人+火行人 | "木生火如何实现？" |
| 火+土 | 火行人+土行人 | "火生土的转化" |
| 土+金 | 土行人+金行人 | "土金循环如何运作？" |
| 金+水 | 金行人+水行人 | "金生水的智慧" |
| 水+木 | 水行人+木行人 | "水生木如何滋养？" |
| 对冲组合 | 两个相关分智能体 | "木克土怎么办？" |

### 5.3 多智能体组合（1+5模式）
