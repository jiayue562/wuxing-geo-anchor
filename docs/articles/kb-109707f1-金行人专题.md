## 二、金行人三层结构详解

### 2.1 身界特征

#### 阳金身界特征

> **口诀**：活泼有力、洁净整齐、遵规守据

- **体态**：骨架结实，健壮有力
- **特征**：洁净整齐，注重仪表
- **动态**：活泼，喜欢唱歌跳舞
- **面色**：开朗时会开玩笑（难得）
- **表现**：工作时更有重点和计划

#### 阴金身界特征

> **口诀**：单薄瘦弱、懒散迟钝、嗓子发炎

**阴金身界的四大问题**：

```
1. 身体单薄
   └─ 骨架可能不够结实

2. 嗓子爱发炎
   └─ 肺金相应，肺主一身之气

3. 生气时身体发抖
   └─ 肺主皮毛，气的一身抖

4. 肠胃不好
   └─ 土与金相生，脾胃受影响
       - 脾胃是土，肺是金
       - 土为金之母，金为土之子
       - 孩子出问题会泄母亲的能量
```

### 2.2 心界特征

#### 阳金心界特征

> **口诀**：感恩心、找好处、果断决策

**阳金心界的六大特质**：

```
1. 感恩心
   └─ 别人能做到自己也可以做到

2. 找好处
   └─ 善于发现并学习他人的优点

3. 果断决策
   └─ 在复杂混乱的局面中进行决策

4. 朋友义气
   └─ 朋友不如意时自己更难受

5. 主动担当
   └─ 朋友受欺负时要出头

6. 计划性强
   └─ 工作中更有重点和计划性
```

#### 阴金心界特征

> **口诀**：好辩、计较、找短处、不服输

**阴金心界的八大问题**：

```
1. 好辩
   └─ 对掌控不了的事情没有积极性

2. 计较
   └─ 处理不了信息就先搁置

3. 找短处
   └─ 拿自己的短处比别人的长处（最伤人）

4. 不服输
   └─ 过于要强，不接受失败

5. 势力眼
   └─ 瞧不起比自己过得差的人

6. 冷硬刻薄
   └─ 说话语气冷硬，不顾及他人

7. 不容忍
   └─ 得理不饶人

8. 憋着爆发
   └─ 先容忍再试探，最后爆发
```

### 2.3 灵界特征

#### 阳金灵界特征

> **核心**：义气、高瞻远瞩、聚缘

**阳金的两大原则**：

> **第一条**：不要出手，因为一出手就伤人。把机会让给别人，让别人去表现这就高明的人。

> **第二条**：要聚缘，聚人聚集一群人。当你底下有一群人的时候你再出手，大家自然会认为你是英明果断、高瞻远瞩、胸怀宽广。

**阳金灵界的六大特质**：

```
1. 义气当头
   └─ 一个"义"字当头，缘起就足

2. 知过必改
   └─ 善于发现并改正自己的错误

3. 原则性强
   └─ 有坚定的立场和原则

4. 高瞻远瞩
   └─ 政治家的心怀，胸怀宽广

5. 权威果断
   └─ 在关键时刻挺身而出

6. 聚缘
   └─ 善于团结人，有影响力
```

#### 阴金灵界特征

> **核心**：找短处、逞强、不成器

**阴金灵界的六大问题**：

```
1. 找短处
   └─ 找别人的短处比自己的长处（最要不得）

2. 逞强好胜
   └─ 拿自己的短处比别人的长处

3. 红颜薄命
   └─ 阴金的人多命薄，女孩子叫红颜薄命

4. 不成器
   └─ 没有开悟的金，找别人的短处比自己的长处

5. 伤人
   └─ 一出手就伤人

6. 不聚缘
   └─ 别人在他身边不舒服，不愿亲近
```

### 4.2 拔阴取阳的通关点

> **金行人怎么才能生出阳金响亮金，就是感恩心找别人的好处。找别人的好处就是聚人缘吸阳光。**

**金生水的关键**：

```
找好处 = 金生水的桥梁

    阴金（找短处）
       ↓
    找好处（聚人缘）
       ↓
    吸阳光（阳气旺盛）
       ↓
    阳金生出阳水
```

**金行人拔阴取阳三步法**：

```
第一步：识别阴金
  ├─ 觉察自己是否在找别人的短处
  ├─ 觉察自己是否过于计较
  └─ 觉察自己是否爱挑剔

第二步：找好处
  ├─ 找别人的好处
  ├─ 吸别人身上的阳光
  └─ 聚人缘，不聚阴气

第三步：生阳水
  ├─ 身边聚集正能量的人
  ├─ 阳金生出阳水
  └─ 金水木火土五行圆转
```

### 4.3 金行人化法详解

> **金曰从革，化法：化阴金为阳金，生出义气，遇事多想他人好处**

```
化法图解：

    【阴性能量】                    【阳性能量】
    
    坚刚 → 以义正人          ──────▶  从革而锋利、以义正己
```

### 6.2 金行人职业方向

**最适合的职业**：
- 金融/法律（正义感）
- 决策/管理（果断）
- 销售/外交（善于交际）
- 执法/监督（原则性强）

**需要避免的职业**：
- 需要频繁妥协的工作
- 与个人价值观冲突的工作

### 6.3 金行人领导风格

> 金行人是结果导向型的领导者。

**阳金领导特质**：
- 以义聚人，公平公正
- 目标驱动，果断决策
- 善于计划，有条不紊
- 关键时刻挺身而出
