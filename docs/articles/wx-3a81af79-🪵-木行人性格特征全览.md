# 🪵 木行人性格特征全览

## 🎯 核心定义

**木行人**，其性格以 **"生长与条达"** 为核心能量，特质围绕 **"独特性、理想主义与内在成长"** 展开。其性格并非静态标签，而是一个动态发展的系统，通过五个核心维度展现，每个维度都包含顺境（健康状态）与逆境（防御状态）下的不同表现，形成一个完整的人格光谱。

**核心特征概括**：
- **生长导向**：所有时间精力都用于内在成长
- **独特性追求**：不够独特就不是木行人
- **理想主义**：透过自我感受认知世界，倾情理想而非现实
- **专家型思维**：深度思考者，战略规划者，创新者


## 🌿 其他相关特征

### （一）身体与外貌特征
1. **形态**：给人的感觉像一个大将军，身材修长、挺拔，如树木般挺拔，骨架偏细，四肢较长，手指纤细。
2. **面色**：黑、白、黄，但一个共同的特性是皮肤不清洁、油腻、发暗。
3. **毛发**：头发浓密、偏硬。
4. **面部**：脸型偏长或瓜子脸，额头较宽，眉毛浓密，眼睛明亮有神。

### （二）能力与事业倾向
1. **团队角色**：是团队中的"创新者"。常常提出一些新想法，这对企业或团队开拓新的思路很有帮助。
2. **专业方向**：属于专家型人才，战略规划者。善用谋略，勤于思辨，学习努力，但体力有限。
3. **事业特征**：木主升发，代表生命的活力，具有条达与柔和两种属性，往往活力四射，运气比较好。

### （三）阴阳两面性总述
**阳面（顺境综合表现）**：
- 有主见，独立性强；正直；肯负责任，有担当力，有才，实干，勤劳，能立。

**阴面（逆境综合表现）**：
- 容易自以为是，做事不许人驳辩，不服人，好抗上；说话不会转弯，态度又直又硬。
