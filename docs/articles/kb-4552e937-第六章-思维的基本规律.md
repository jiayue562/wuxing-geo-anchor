## 一、同一律

- **内容**：在同一思维过程中，概念或命题必须保持同一，不能偷换概念、转移论题
- **公式**：A是A
- **要求**：①概念内涵外延不变 ②命题含义不变
- **违反**：偷换概念、混淆概念、转移论题
- **例**："我从小爱运动（体育运动）；中学学物理，运动（物理运动）是物体位置变化，所以运动很重要"——偷换概念


## 三、排中律

- **内容**：两个互相矛盾的命题不能同假，必有一真
- **公式**：要么A，要么非A
- **要求**：对矛盾命题必须明确表态，不能"两不可"
- **适用范围**：只适用于矛盾命题，不适用于反对命题
- **违反**：模棱两可


## 五、四律的层次关系

```
同一律 + 不矛盾律 + 排中律  →  思维形式的基本规律（保障思维的确定性）
充足理由律                   →  论证的基本规律（保障论证的可靠性）
```
