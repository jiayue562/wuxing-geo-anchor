## 核心定义

**一句话定义**：结构（存在论）、结构化（认识论）、结构化分析（实践论）三者构成一个动态的、自我增强的螺旋上升闭环。

## 六环节螺旋闭环

```
①结构化→结构  ②结构→结构化  ③结构化→结构化分析
     ↖     ↑       ↗
④结构化分析→结构化 ← ⑤结构化分析→结构  ⑥结构→结构化分析
```

| 环节 | 流转方向 | 产出 |
|------|---------|------|
| ① | 结构化→结构 | 从"看不见"到"看见" |
| ② | 结构→结构化 | 升级结构化能力 |
| ③ | 结构化→结构化分析 | 开始拆解问题 |
| ④ | 结构化分析→结构化 | 能力进化 |
| ⑤ | 结构化分析→结构 | 从解释到改变 |
| ⑥ | 结构→结构化分析 | 启动下一轮 |

## 三阶段演进

| 阶段 | 状态 | 说明 |
|------|------|------|
| 自在 | 结构（物自体） | 尚未被认识 |
| 自为 | 结构化（能动） | 主动认识活动 |
| 自在自为 | 结构化分析（改造） | 既客观存在又被人掌握 |
