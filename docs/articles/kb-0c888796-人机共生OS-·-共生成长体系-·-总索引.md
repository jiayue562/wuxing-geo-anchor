### 2.2 AEP系统

**核心公式**：
```
AI总成长度 = EXP贡献 + AEP贡献
EXP贡献 = floor(累计EXP / 100) × 1
AEP贡献 = floor(累计AEP / 80) × 1
D值 = 基础D值 + EXP贡献 + AEP贡献
```
