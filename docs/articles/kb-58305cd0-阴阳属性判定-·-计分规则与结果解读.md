## 一、计分方法

### 1.1 评分标准
每题 5 分制：1=完全不符合，5=完全符合

### 1.2 维度独立计分
每个维度（基本属性、亲密关系、团队领导力）独立计分
- 阳/阴各自加总
- 某行总分 = 阳分 + 阴分

### 1.3 主导判定
- 最高分行 = 主导五行
- 次高分 = 辅助五行
- 阴阳比 = 阳分/阴分（1.2 倍为阈值）
