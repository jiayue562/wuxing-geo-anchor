## 🎯 为何是"五"？—— 数的逻辑必然性

### 数学结构证明

**李约瑟洞见**: 五行"并不是五种基本物质，而是五种基本的程序"

**形式逻辑证明**:
```
要满足以下条件：
1. 每一元素都能生另一元素
2. 每一元素都能克另一元素
3. 任何两元素间的关系唯一

→ 元素数量必须为五

证明：
- 四元素：无法形成完整的生克循环
- 五元素：外圈相生 + 内圈相克，双向互动网络
- 六元素：关系冗余，破坏唯一性
```

### 系统论意义

| 特性 | 五行表达 | 现代系统论对应 |
|------|----------|----------------|
| 自组织 | 相生循环 | 正反馈机制 |
| 自平衡 | 相克循环 | 负反馈机制 |
| 最小充分单元 | 五元素 | 复杂度阈值 |
| 封闭又开放 | 循环但互动 | 耗散结构 |

**核心洞见**: "五"是构建具有自组织、自平衡特性关系网络的最小充分单元。

## 🔄 生克关系总览

### 相生循环（正向动力链）

```
木 → 火 → 土 → 金 → 水 → 木
│    │    │    │    │
│    │    │    │    └── 水生木（滋润涵养）
│    │    │    └── 金生水（沉降凝聚）
│    │    └── 土生金（凝聚沉淀）
│    └── 火生土（能量化灰）
└── 木生火（生长提供燃料）
```

**应用原则**: "补母生子"（虚则补其生源）、"顺势而为"


### 生克反常与转化

| 异常类型 | 定义 | 示例 | 调理方法 |
|----------|------|------|----------|
| **相乘** | 过度相克 | 木亢乘土 | 培土生金以制木 |
| **相侮** | 反向相克 | 木虚侮金 | 滋水涵木以平金 |
| **反生** | 生者无力 | 金弱不生水 | 补金生水 |
| **反克** | 被克反胜 | 土极盛反侮木 | 疏土扶木 |
