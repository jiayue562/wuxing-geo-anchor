## 六、使用指南

### 6.1 如何阅读知识图谱

1. **按领域查找**：查看特定领域的知识联系
2. **按节点追踪**：从核心节点出发追踪关联
3. **按标签筛选**：使用标签快速定位相关内容
4. **按权重排序**：优先关注高权重的联系

### 6.2 如何扩展知识图谱

1. **添加新节点**：发现新的核心概念
2. **建立新联系**：连接已有节点
3. **更新权重**：根据实践调整联系强度
4. **补充标签**：完善标签体系
