```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│    入口（精准线上引流）                                       │
│          ↓                                                   │
│    客户池（精准人群：刺身爱好者）                               │
│          ↓                                                   │
│    ┌──────────────────────────────────────────────────┐     │
│    │                                                      │     │
│    │  核心转化 → 储值会员（最强粘性）                       │     │
│    │                                                      │     │
│    │  次要转化 → 普通微会员（日后升级）                       │     │

│          ↓                                                   │
```

| 节点 | 目标 | 策略 |

## 六、价值提升

### 6.1 提升策略

| 维度 | 策略 | 目标 |
|------|------|------|
| **堂食+酒水** | 组合推荐 | 高客单 |
| **外卖** | 增加消费频次 | 多场景 |

### 6.2 价值最大化

- **高客单**：通过堂食+酒水组合
