## 二、五行人职业性格特征

### 1. 木行人职业性格特征

| 阳性特质 | 阴性特征 |
|----------|----------|
| + 逻辑性强的 | - 关注细节的 |
| + 放松、安静的 | - 冷淡、抗上 |
| + 独特、有思想的 | - 忧愁、不服 |
| + 理想主义 | - 不喜风险的 |
| + 在习惯性领域 | - 节奏缓慢的 |

**阳性行为**：
- 自认天生与众不同，衣着言行均有个人风格，拒绝平庸
- 善于并乐于通过难题展示才华
- 追求专业深度，知识广博，见解独到
- 经常产生有创意的办法和思路
- 喜欢从事帮助他人的工作，行事温和儒雅

**阴性行为**：
- 服饰装扮玩酷，重视视觉刺激，生活在自己的思维世界里
- 在擅长的领域孤芳自赏，介意别人超越自己
- 愿意代表专业的权威发表言论，不屑接受他人意见
- 敢于挑战权威，认为领导不够专业
- 总认为公司没有给自己足够空间施展才华

### 2. 火行人职业性格特征

| 阳性特质 | 阴性特征 |
|----------|----------|
| + 积极主动 | - 情绪多变的 |
| + 鼓舞、热情的 | - 急躁、苛求的 |
| + 健谈好客的 | - 夸大、炫耀的 |
| + 精力充沛的 | - 虚荣、好争的 |
| + 以人驱动目标任务 | - 超级明星 |

**阳性行为**：
- 讲话底气足，声音洪亮，追求完美，穿着得体
- 善于制定准则，要求大家认同跟随
- 对社会道德正义有是非感，自己严格遵守
- 激励的、乐观的、良好的协调者
- 敢担风险，挑战空间越大越敢于接受

**阴性行为**：
- 对外表特别注重，色彩鲜艳，讲究排场
- 好争理，缺乏耐心，总觉得自己是真理的化身
- 好面子，情感世界薄弱，害怕被否定
- 工作总希望别人服从自己的决定
- 言语苛求，对人指责，态度严肃

### 3. 土行人职业性格特征

| 阳性特质 | 阴性特征 |
|----------|----------|
| + 以团队为中心 | - 目标不突出 |
| + 具有合作精神 | - 保守、固执 |
| + 诚信，助人的 | - 沉闷、迟钝 |
| + 有责任感 | - 疑心重 |
| + 以资源整合为优势 | - 不自信 |

**阳性行为**：
- 衣着简朴，不铺张浪费，举止稳健，言语诚实
- 总能为其他流程的工作着想，勇于承担责任
- 不喜结交权贵，与伙伴齐心同力，群众基础好
- 勤恳实干，稳定持久，恶劣环境依然从容
- 包容量大，待人友善，为人忠诚

**阴性行为**：
- 不重外表，说话声音单调，神采不足
- 缺乏计划与监控，重点不突出，缺乏创新
- 对权威的领导相当不信任，表现出顺从又怀疑
- 不自信，放大困难，好怨人
- 接受新鲜事物的能力慢，容易落伍

### 4. 金行人职业性格特征

| 阳性特质 | 阴性特征 |
|----------|----------|
| + 刚毅果断的 | - 冷酷、集权的 |
| + 有远见的 | - 忽略他人的 |
| + 注意力集中 | - 计较、比较的 |
| + 有管理组织能力 | - 自负的 |
| + 以目标为导向 | - 小气的 |

**阳性行为**：
- 着装正式，符合成功人士的形象
- 能力出色，鞠躬尽瘁地工作，常怀感恩心态
- 严格遵守职业操守，以身作则
- 独立高效，行动有计划，善于制定目标及执行
- 努力工作的实干家，天生具有直觉，多才多艺

**阴性行为**：
- 有竞争意识，支配他人的愿望强
- 不断追逐下一个目标，拜金主义，讲究物质享受
- 是典型的工作狂，容易紧张，回避感情
- 以自我为中心，认识刻板、固执，缺乏灵活
- 不喜欢和能力差的同事交流，认为浪费时间

### 5. 水行人职业性格特征

| 阳性特质 | 阴性特征 |
|----------|----------|
| + 智慧的 | - 松懈怠工的 |
| + 善解人意的 | - 懒散、混乱 |
| + 沉稳、安静的 | - 消沉、抑郁 |
| + 和谐团建的 | - 缺乏安全感 |
| + 以情感为导向的 | - 意志薄弱 |

**阳性行为**：
- 追求自然平和，不注重衣服款式，但注重质地
- 为人真实，待人真诚，办事真切
- 办事灵活，适应性强，城府较深
- 认为工作要快乐，只有快乐的工作才是真正的工作
- 助人付出，不求回报

**阴性行为**：
- 随遇而安，不喜冲突，工作不求上进
- 言语富有感情色彩，喜欢发展私人情感
- 关注关系情感胜过目标任务，承担压力能力低
- 为了维护关系自己承担太多付出，容易选择逃避
- 受气包，爱烦人，爱哭，消沉抑郁

### 解读

**一眼定乾坤**：先看一个人的格局
- 木"瘦"、火"尖"、土"厚"、金"方"、水"肥"

**两眼醍醐灌顶**：再看一个人的局部
- 木行人"三长"：脸长、身材四肢长、手指长
- 火行人"三尖"：头顶尖、鼻头尖、下颚尖
- 土行人"三厚"：身材短、胸背厚、手背厚
- 金行人"三薄"：嘴唇薄、眼皮薄、手背薄
- 水行人"三肥"：眼皮肥（多层）、下巴肥、手背肥


## 六、创新整合：拔阴取阳行为操作指引

基于职业特征阳阴对比，可以为每个五行创建**行为层面的拔阴取阳操作指引**：

### 木行人

| 阴木行为 | 转化路径 | 阳木行为 |
|---------|---------|---------|
| 在擅长领域孤芳自赏 | → 学 | 乐于分享，共同成长 |
| 挑战权威，认为领导不够专业 | → 仁 | 尊重他人，善用其长 |
| 总认为没有足够空间施展 | → 生 | 创造机会，主动作为 |

### 火行人

| 阴火行为 | 转化路径 | 阳火行为 |
|---------|---------|---------|
| 好争理，总觉得自己是真理 | → 礼 | 明辨是非，尊重不同 |
| 要求别人服从自己的决定 | → 明 | 接纳建议，灵活调整 |
| 讲究排场，追求外表 | → 稳 | 注重内在，稳重务实 |

### 土行人

| 阴土行为 | 转化路径 | 阳土行为 |
|---------|---------|---------|
| 缺乏计划，重点不突出 | → 信 | 明确目标，专注执行 |
| 怀疑一切，不自信 | → 思 | 理性分析，坚定信念 |
| 接受新事物慢，容易落伍 | → 厚 | 开放心态，持续学习 |

### 金行人

| 阴金行为 | 转化路径 | 阳金行为 |
|---------|---------|---------|
| 拜金主义，追逐物质 | → 义 | 合理追求，适度享受 |
| 以自我为中心，固执 | → 行 | 倾听他人，灵活变通 |
| 挑剔计较，缺乏包容 | → 净 | 心胸开阔，宽容待人 |

### 水行人

| 阴水行为 | 转化路径 | 阳水行为 |
|---------|---------|---------|
| 工作不求上进，得过且过 | → 智 | 明确目标，积极进取 |
| 关注关系胜过任务 | → 辩 | 平衡关系与目标 |
| 意志薄弱，容易逃避 | → 柔 | 坚韧不拔，勇于担当 |
