## 二、理论根基溯源

### 2.1 象思维根基

**核心原理**：取象比类——自然之象 → 人身之象

**自然之象的三大属性**：
```
水无常形 → 身圆（形态圆润）
水往低处流 → 形垂（沉潜下沉）
水能滋养万物 → 肤松（温润泽被）
```

**坎卦（☵）体系**：
```
坎卦 = 水
├── 卦象：两阴夹一阳（外柔内刚）
├── 尚秉和《周易尚氏学》：
│   ├── "坎为圆" → 身圆
│   ├── "坎为润" → 肤松
│   └── "坎为沉" → 形垂
└── 对应落地：
    ├── 身圆 = 坎体之圆
    ├── 肤润 = 坎性之泽
    └── 沉潜 = 坎德之沉
```

## 四、细节特征解析

### 4.1 头面特征

| 部位 | 特征描述 | 识别要点 |
|------|---------|---------|
| **头型** | 大且圆 | — |
| **脸型** | 圆形或椭圆，面部饱满（澎皮） | 按压能出坑 |
| **眼睛** | 有神，多为双眼皮，眼皮肥厚 | 判断阴阳关键 |
| **鼻子** | 鼻梁不高，鼻头圆润有肉 | 缺乏金行人之挺拔 |
| **耳朵** | 大多有耳垂，形态圆润 | 耳垂对应肾气 |
| **眉毛** | 不特别浓密，偏淡 | 清秀或温顺感 |

### 4.2 躯干与四肢

**"三肥"定位法**：
```
眼皮肥 → 第一肥（眼周）
下巴肥 → 第二肥（下巴）
手背肥 → 第三肥（手背）
    ↓
三者皆备 → 水行人无疑
```

### 4.3 肤色与毛发

| 项目 | 特征描述 | 状态判断 |
|------|---------|---------|
| 基础色 | 黑色或偏黑 | 五行水色投射 |
| 变体色 | 偏白润 | 取决于肾水代谢 |
| 阳水面色 | 亮泽，黑而亮如重漆 | 健康 |
| 阴水面色 | 晦暗如地苍/苍白无华 | 肾气衰败 |

## 八、实操识别体系

### 五步标准化识别

#### 步骤一：远观轮廓（定体型）
```
距离：3-5米
观察点：
├── 是否呈"A"字型？
├── 肩膀是否较窄？
├── 臀部是否宽厚圆润？
└── 重心是否偏低？
判断：轮廓圆润 + 重心偏低 → 水行倾向
```

#### 步骤二：近察肤质（定质感）
```
距离：1米内
观察点：
├── 皮肤是否偏松弛？
├── 眼皮是否多层或肥厚？
└── 触感：软滑？温润？回弹慢？
```

#### 步骤三：细辨五官（定特征）
```
观察点：
├── 下巴是否偏垂且圆润？
├── 眼神是否温和深邃？
└── 嘴唇是否偏厚？
```

#### 步骤四：动静结合（定神韵）
```
动态观察：
├── 行走时是否摇摆、迟缓？
├── 站立时是否喜欢倚靠？
└── 声音是否柔和、语速平缓？
```

#### 步骤五：阴阳定性（定状态）
```
阳水：眼神通透 + 面色润泽 + 体态丰腴
阴水：眼神迷茫 + 面色晦暗 + 体态臃肿
```

> "木瘦金方水必肥，上尖下阔为真火，土行背厚晃如龟，朝中方白属金行。"

```
一眼看眼皮（多层/肥厚）

三者皆备 → 水行人无疑
```
