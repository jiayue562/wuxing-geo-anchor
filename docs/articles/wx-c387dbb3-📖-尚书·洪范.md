# 📖 尚书·洪范

## 核心定义

《尚书·洪范》是五行识人的**经典源头**，长期被忽视却蕴含**完整象系统**。

**核心地位**：完成**象→辞第一次抽象**，是五行识人的奠基之作。

## 洪范九畴

《洪范》九畴把五行纳入治国大法，提出**原象词**：

> **"水曰润下，火曰炎上，木曰曲直，金曰从革，土爰稼穑"**

这五个原象词成为后世一切五行阐释的**基因**。

## 五行原象词详解

| 五行 | 原象词 | 能量态势 | 核心特征 |
|------|--------|---------|---------|
| 木 | 曲直 | 生发、舒展 | 如木之生长，曲折有度 |
| 火 | 炎上 | 炎上、扩散 | 如火之燃烧，向上升腾 |
| 土 | 稼穑 | 承载、化育 | 如土之生长，播种收获 |
| 金 | 从革 | 收敛、坚锐 | 如金之变革，顺从有度 |
| 水 | 润下 | 润下、潜藏 | 如水之流动，向下渗透 |
