### 2.1 诊断工具链

```
三维诊断模型
├── 物质层 → 文化健康度评估量表
├── 能量层 → 差序格局关系图谱
├── 信息层 → 五行识人 + 五色光价值观测评
│
└──→ 七维企业文化评估量表
    ├── 主导特征 / 组织领导力 / 员工管理
    ├── 企业凝聚力 / 战略要点 / 成功标准 / 冲突管理
    └──→ 五类型：A创新 / B活力 / C和谐 / D规则 / E家庭
```

### 2.2 行为经济学落地链

```
行为经济学双锚模型
├── 锚1：损失厌恶（2.5倍痛苦感）
│   ├── 微观干预：预算精确目标 / 容错机制 / 创新风险共担 / 福利阶梯
│   └── 伦理边界：透明知情 + 尊重自主 + 长期测量
│
├── 锚2：锚定效应（初始信息依赖）
│   ├── 文化锚定：理念体系锚定 / 价格价值定位 / 行为规范锚定
│   └── 仪式感 = 行为锚定的文化编码
│
└──→ 选择架构（Choice Architecture）
    └── 礼治 ≈ 最古老的行为经济学工具
```

### 2.3 AI赋能四层架构

```
AI+企业文化四层架构
├── L1 提示词 → 生产力重构
│   └── SKILL.md = 提示词的工程化载体
│
├── L2 知识库 → 独特性重塑
│   └── LLM Wiki + RAG = 知识的持久化与检索
│
├── L3 工作流 → 流程自动化
│   └── 龙爪信号协议 = 工作流的标准化传输
│
└── L4 智能体 → 组织智能化
    └── 岗位智能体 = 智能体的工程实现
    └──→ 去总部化的技术基础
```
