### 4.1 触发机制

```yaml
# 工具层模型触发规则

触发条件:
  - 用户明确提到具体模型名称
  - 用户请求"使用XX模型分析"
  - 用户说"用XX思维"

加载策略:
  - 单模型：直接加载对应模型
  - 多模型组合：并行加载，组合输出
  - 场景套件：加载整个场景的5-6个模型

响应时间:
  - 单模型：<0.5秒
  - 多模型：<1秒
  - 场景套件：<2秒
```

## 六、与上层的协同

### 6.1 三层联动

```
核心层（9个）：
  - 感知即激活
  - 直接穿透本质
  ↓ 组合使用
模式层（20个）：
  - 场景识别后激活
  - 系统性分析
  ↓ 精准补充
工具层（71个）：
  - 指令调用
  - 深度专项分析
```

### 6.2 组合使用示例

```
场景：新产品市场进入策略

核心层激活（自动）：
  → 第一性原理：市场本质是什么？
  → SWOT：四维分析
  → 五色光：多维透视

模式层激活（场景识别）：
  → 决策分析模式：风险评估
  → 创新突破模式：突破口

工具层调用（指令）：
  → 麦肯锡战略套件：战略定位
  → 麦肯锡市场套件：用户画像
  → 财务投资套件：ROI分析
```
