## 核心定义

**五行人格心理学**是将中国传统五行理论（木火土金水）与现代心理学相结合的人格分析体系。


# 🌏 土行人完整体系

> **核心定义**：土行人以「承载觉知」为一心，具有承载性与稳定性的双重特质。如大地厚德载物，土行人天生具备包容力、落地力和信实承载的使命，是执行者、稳定者和组织的压舱石。


## 🌍 二、三界：土象显化

| 三界 | 阳土显化（健康） | 阴土失衡（不健康） |
|------|----------------|-------------------|
| **身界** | 体型圆润，动作稳重，面色黄润，声音沉稳 | 体型臃肿或虚胖，动作拖沓，脾胃不和，易消化问题 |
| **心界** | 情绪以信任、包容、感恩为主；思维踏实落地 | 情绪以怨恨、疑虑、担忧为主；思维保守固执 |
| **灵界** | 价值观核心是"信"——信实可靠、厚德载物；使命"承载与滋养" | 价值观扭曲为"疑"——不信任他人；使命扭曲为"控制与固守" |


## 📊 四、九层：土行发展光谱

| 区间 | 层级 | 核心状态 | 特征 |
|------|------|---------|------|
| **健康** | 1级 | 信实圆融，自在承载 | 厚德自然流露，无求滋养 |
| | 2级 | 稳重落地，有效执行 | 信实使命驱动，善于落地 |
| | 3级 | 踏实可靠，价值体现 | 在社会角色中展现稳定价值 |
| **一般** | 4级 | 怨恨初现，内外拉扯 | 疑虑出现，稳定性下降 |
| | 5级 | 拖延消极，怨天尤人 | 使命扭曲为"等待他人改变" |
| | 6级 | 固执守旧，过度补偿 | 用守旧来维持安全感 |
| **不健康** | 7级 | 怨恨攻击，被动抵制 | 消极怠工，关系破坏 |
| | 8级 | 妄想强迫 | 三界协同断裂 |
| | 9级 | 病态固守 | 系统崩解 |


## 🔧 六、土行人转化技术

### 拔阴取阳口诀

| 阴土状态 | 转化口诀 | 阳土状态 |
|---------|---------|---------|
| 怨（怨恨不满） | **阴土怨转阳土信** | 信实承载·感恩包容 |
| 疑（疑虑不信） | 疑→信（建立信任） | 信实笃定·包容接纳 |
| 拖（拖延消极） | 拖→行（立即行动） | 踏实落地·稳步推进 |

### 土行通关点
**感恩**：对土行人表达真诚感谢，让他感到付出是有价值的
- "欲培真土必找好处"（感恩=找好处的升级版）
- 认可土行人的付出，触发土行人的信实能量
