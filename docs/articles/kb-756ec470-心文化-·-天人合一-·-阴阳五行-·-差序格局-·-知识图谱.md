### 4.1 费孝通理论

> 我们的格局不是一捆一捆扎清楚的柴，而是好像把一块石头丢在水面上所发生的一圈圈推出去的波纹。

### 4.2 差序格局结构

```
                         己（中心）
                          │
            ┌─────────────┼─────────────┐
            │             │             │
            ▼             ▼             ▼
        ┌───────┐    ┌───────┐    ┌───────┐
        │ 近亲  │    │ 朋友  │    │ 邻里  │
        │ (强)  │    │ (中)  │    │ (弱)  │
        └───┬───┘    └───┬───┘    └───┬───┘
            │             │             │
            │   波纹一圈圈推开          │
            │             │             │
            └─────────────┼─────────────┘
                          │
                          ▼
                    ┌───────────┐
                    │ 共同体    │
                    │ (差序网络)│
                    └───────────┘
```

### 4.3 差序格局 vs 团体格局

| 维度 | 差序格局（中国） | 团体格局（西方） |
|------|----------------|-----------------|
| 结构比喻 | 石子投入水面，波纹推开 | 一捆一捆扎清楚的柴 |
| 边界 | 模糊，伸缩自如 | 清晰，成员平等 |
| 道德基础 | 私人联系中有意义 | 普遍主义 |
| 组织逻辑 | 以"己"为中心 | 以"团体"为中心 |
| 文化代表 | 费孝通 | 西方社会学 |
