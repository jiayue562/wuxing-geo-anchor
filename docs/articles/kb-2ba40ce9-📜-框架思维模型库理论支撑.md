### 1.2 心理学基础：双过程理论

框架思维模型库整合了**双过程理论**（Kahneman, 2011）：

| 过程 | 系统1（快思考） | 系统2（慢思考） |
|------|----------------|----------------|
| **特点** | 自动、迅速、无意识 | 费力、缓慢、有意识 |
| **对应概念** | 思维框架（肌肉记忆） | 框架思维（向导视角） |
| **优势** | 高效、节能 | 审慎、精准 |
| **风险** | 偏见、锚定 | 懒惰、过度分析 |
| **转化路径** | 系统1熟练 → 系统2觉醒 → 新系统1形成 | |

**理论整合**：
> 框架思维的本质，是让「系统2」帮助「系统1」升级——不是消灭直觉，而是让直觉更精准。


## 二，理论框架详析

### 2.1 框架理论

#### 2.1.1 框架的认知科学基础

**定义**（来源：Tversky & Kahneman, 1974）：
> 框架是决策者用来组织和解释信息的心理结构。它决定了哪些选项被考虑，哪些被忽略。

**认知功能**：
1. **简化功能**：降低认知负荷
2. **组织功能**：赋予信息以结构
3. **筛选功能**：决定注意力的分配
4. **预测功能**：提供因果推断的模板

#### 2.1.2 框架建构三要素（来源：《框架思维》）

**因果律（Cause-Effect）**：
- 为事物建立「如果…那么…」的叙事逻辑
- 强大的框架能提供简洁有力的因果故事
- 作用：帮我们在复杂中快速定位「谁在起作用」

**反事实思维（Counterfactual）**：
- 在头脑中虚拟「并非如此却可能如此」的情景
- 作用：打破默认路径，构想替代方案，拓宽可能性边界

**约束条件（Constraints）**：
- 为思考划定边界
- 决定「什么可以纳入，什么必须排除」
- 作用：好的约束不是限制，而是聚焦


### 2.3 框架思维理论

#### 2.3.1 元认知理论

**定义**（来源：Flavell, 1979）：
> 元认知是「关于认知的认知」——它包括元认知知识、元认知体验和元认知监控。

**框架思维的元认知结构**：

| 元认知成分 | 框架思维表现 |
|-----------|-------------|
| **元认知知识** | 知道不同框架的特点、适用场景 |
| **元认知体验** | 意识到当前框架可能不适用的「不对劲」感 |
| **元认知监控** | 主动审视「这个框架合适吗？」 |

#### 2.3.2 认知灵活性理论

**定义**（来源：Spiro, Coulson, Feltovich, Anderson, 1988）：
> 认知灵活性是指根据情境需要，在不同知识表征之间灵活转换的能力。

**框架思维的灵活性层级**：
1. **选择性**：从众多框架中选择最合适的
2. **组合性**：将多个框架嵌套使用
3. **创造性**：当无现成框架时，创造新框架


### 3.2 系统思维理论

#### 3.2.1 系统冰山模型

**层次结构**（来源：Donella Meadows《系统之美》）：

```
         ┌─────────────────────────┐
         │      事件层（Event）     │ ← 发生了什么
         └────────────┬────────────┘
                      │
         ┌────────────┴────────────┐
         │     模式层（Pattern）     │ ← 有没有重复发生
         └────────────┴────────────┘
                      │
         ┌────────────┴────────────┐
         │     结构层（Structure）   │ ← 什么系统结构导致
         └────────────┴────────────┘
                      │
         ┌────────────┴────────────┐
         │     心智层（Mental）     │ ← 什么信念维持结构
         └─────────────────────────┘
```

#### 3.2.2 反馈回路理论

**正反馈回路（Reinforcing Loop）**：
> 增强初始变化的影响——A增强B，B反过来增强A

**负反馈回路（Balancing Loop）**：
> 寻求平衡——当系统偏离目标时，自动纠正

**延迟效应**：
> 原因与结果之间存在时间差——需要拉长时间轴审视

### 4.1 问题解决理论

#### 4.1.1 问题空间理论

**定义**（来源：Newell & Simon, 1972）：
> 问题解决是从初始状态，通过一系列操作符，达到目标状态的过程。

**问题类型**：

| 问题类型 | 特征 | 框架选择 |
|----------|------|----------|
| **良构问题** | 目标明确、状态清晰 | 结构化框架 |
| **劣构问题** | 目标模糊、利益多元 | 框架思维+创造性 |

#### 4.1.2 问题解决过程模型

**阶段模型**：

## 五，跨域理论整合

### 5.1 与象思维的同构

| 象思维层次 | 框架思维对应 | 本质 |
|------------|-------------|------|
| **物象层** | 框架（地图） | 具体的、可操作的工具 |
| **意象层** | 思维框架（肌肉记忆） | 内化的、习惯性的思维模式 |
| **原象层** | 框架思维（向导视角） | 超越形式的、智慧的选择能力 |

**同构原理**：
> 框架的三层进化（框架→思维框架→框架思维）本质上是象思维在认知领域的展开。
