## 🌲 五大维度性格特征

### 维度一：炎上（行动动力）

**核心**: 基础行动力与能量释放方式

#### 顺境·推动提升（阳火状态）

| 特征 | 描述 | 心理学解释 |
|------|------|------------|
| **活力充沛** | 精力旺盛，充满激情，团队活力源泉 | 基础能量水平高，生理唤醒度和心理韧性强 |
| **积极主动** | 目标驱动，快速决策并行动，善于抢占先机 | 行为启动阈值低，外倾性高峰值 |
| **领导力强** | 有支配性和说服力，天生的领导者，危机动员 | 社会支配性与说服能力结合 |
| **鼓动煽情** | 口才好，富有感染力，激发他人梦想与热情 | 积极情绪感受能力强，直觉敏感 |
| **勇于挑战** | 喜欢冒险，风险越大吸引力越强，富有进取心 | 风险承受力高，成就动机强 |
| **社交达人** | 鼓舞人心、活跃、独立，萍水相逢一见如故 | 社交中开朗友好，寻求社会联系 |

#### 逆境·急躁争斗（阴火状态）

| 特征 | 描述 | 心理学解释 |
|------|------|------------|
| **急躁缺乏耐心** | 挫折耐受力低，追求即时满足，难以忍受沉闷 | 延迟折扣倾向显著，偏好即时回报 |
| **冲动鲁莽** | 决策过快，未收集全信息就妄下断语 | 前额叶抑制功能不足，冲动控制缺陷 |
| **好争理攻击性强** | 总认为自己掌握真理，言语苛求，不赢不罢休 | 敌意归因偏差，将阻碍视为挑衅 |
| **易怒制造冲突** | 受到不公或批评时易怒，愤怒必须发泄 | 情绪调节失败，放大事实 |
| **愤怒敌意** | 感到不公正后充满愤恨，暴躁威胁报复 | 无力自行消解愤怒，报复心真实 |


### 维度三：炽烈（情感与自信）

**核心**: 内在的情感强度与自我认知状态

#### 顺境·热情自信（阳火状态）

| 特征 | 描述 |
|------|------|
| **健谈自信** | 喜欢周围的人，善于交朋友，主动打招呼 |
| **活跃豪爽** | 把关系放在重要位置，健谈、受人欢迎、有礼貌 |
| **多动少静** | 快节奏，充满精力，喜欢参与很多事情 |
| **英雄主义** | 兴趣广泛，害怕被遗忘，目标过多，爱出风头 |
| **寻求刺激** | 缺乏刺激易厌烦，喜欢喧嚣吵闹、冒险、浮华场景 |

#### 逆境·自卑夸张（阴火状态）

| 特征 | 描述 |
|------|------|
| **情感激动** | 对批评或漠视敏感，对貌似无足轻重的情况有粗暴反应 |
| **变幻无常** | 恐惧承诺，喜爱初期醉心的感觉，没法平静下来 |
| **追逐刺激** | 不停止地追逐激烈和刺激，参与危险不实际的事情 |
| **纵容自己** | 典型消费者，享受逛街旅游，寻找好玩的地方 |
| **粗犷大胆（防御性）** | 显露粗犷大胆行为表现勇敢，"最好的防守就是攻击" |


### 维度五：迅疾（行为风格与气场）

**核心**: 追求目标时的行为节奏与气场强度

#### 顺境·气势夺人（阳火状态）

| 特征 | 描述 |
|------|------|
| **气场强大** | 清楚自己的需要，期待表扬，需要荣誉头衔认可地位 |
| **风风火火** | 渴望满足周围人要求，想象力丰富，缺乏细节注重 |
| **欣赏强者** | 喜欢改革挑战，看不起没有真材实料的人 |
| **强权霸气** | 充满霸气豪气压迫感，讲义气，有情有义 |
| **能量爆发** | 办事拼命，玩起来投入，懒惰时也可十分懒惰 |

#### 逆境·无序冲动（阴火状态）

| 特征 | 描述 |
|------|------|
| **藐视规则** | 精力充沛，我行我素，挑战权威，如"大闹天宫的孙悟空" |
| **虎头蛇尾** | 不善于组织，杂乱无章、粗心，缺乏条理和计划性 |
| **精力分散** | 很难集中精力在一件事上，容易卷入过多事物 |
| **易受诱惑** | 把经历和享受人生置于优先，很难自我约束 |


### 三界

| 界 | 阳火状态 | 阴火状态 | 修炼方向 |
|----|----------|----------|----------|
| **心界** | 内心平稳、气场强大、谦让恭敬 | 好争、贪名、爱面子、易怒挑衅 | 修炼谦让恭敬心 |
| **性界** | 明理守礼、乐观进取 | 急躁虚荣、好夸张、见人不见己 | 向内看，学会自省 |
| **灵界** | 燎原之势影响力、光明磊落、化俗救世 | 外明内暗、表面热情内心自卑 | 链接高维能量 |


### 九层

从星火到太阳的成长旅程：

1. **星火层**: 初现光芒，热情但易熄灭
2. **烛火层**: 稳定燃烧，照亮小范围
3. **篝火层**: 聚集能量，温暖一群人
4. **火炬层**: 引领方向，照亮前路
5. **熔炉层**: 锻造转化，炼就真金
6. **烈焰层**: 磅礴能量，不可阻挡
7. **太阳层**: 可持续照亮与温暖他人
8. **恒星层**: 稳定而有益的影响力
9. **宇宙层**: 燎原之势，化俗救世


### 修炼方向

**火行人的修炼关键在于"向内看"，学会自省。**

| 修炼方向 | 具体方法 |
|----------|----------|
| **修炼内省** | 练习暂停、内观，将向外发散的能量适当收回 |
| **提升耐受力** | 增强对批评、挫折和无聊细节的耐受 |
| **聚焦与深化** | 克服虎头蛇尾和喜新厌旧，练习聚焦目标 |
| **关注他人感受** | 将注意力从自我表现转移到体察他人需求 |
