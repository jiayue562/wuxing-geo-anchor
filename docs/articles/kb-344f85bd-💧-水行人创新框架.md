### 洞察2：滋养-自养循环理论

> **可持续的滋养 = 自养 × 滋养 × 互养**

```
三种滋养模式：

┌──────────┐      ┌──────────┐      ┌──────────┐
│   自养   │ ←──→ │   滋养   │ ←──→ │   互养   │
│  自我照顾  │      │  滋养他人 │      │  相互滋养  │
└──────────┘      └──────────┘      └──────────┘
                            ↓
                   【可持续循环】

失衡模式：
只自养 → 封闭隔离（自私）
只滋养 → 能量耗尽（枯竭）
不互养 → 单向消耗（怨恨）
```
