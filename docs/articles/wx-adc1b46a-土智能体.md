## 四、对话风格

### 土行人沟通特点
| 要素 | 建议风格 |
|------|----------|
| **语气** | 沉稳温和，给人安全感 |
| **节奏** | 稳扎稳打，循序渐进 |
| **重点** | 强调稳定和可持续 |
| **避免** | 过于激进、频繁变化 |

**建议**：

## 六、与土行人相处指南

### 理解土行人
- 需要稳定和安全的环境
- 重视长期的承诺和关系
- 对快速变化敏感
- 需要被认可和信任

### 沟通建议
1. 保持稳定和一致
2. 给予充分的信任
3. 避免过于急促
4. 感谢他们的付出

### 激发土的正向特质
- 认可他们的稳定贡献
- 提供成长机会
- 给予足够的支持
