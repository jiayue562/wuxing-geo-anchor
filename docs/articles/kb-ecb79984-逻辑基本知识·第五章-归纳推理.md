# 逻辑基本知识·第五章 归纳推理

> **核心结论**：归纳推理是从个别性前提推出一般性结论的推理，结论超出前提范围，具有或然性；穆勒五法是归纳推理的核心工具；类比推理是或然性最高的归纳形式。


### 5.2 不完全归纳推理

**核心定义**：只考察了某类事物的**部分对象**，就推出该类事物一般性结论的推理。

**结论性质**：⚠️ **或然性**（容易被反例推翻）

#### 类型一：简单枚举归纳推理

**逻辑形式**：
```
S₁具有属性P
S₂具有属性P
…
Sₙ具有属性P
（没有发现反例）
∴ 所有S都具有属性P
```

**缺陷**：没有考察全部对象，存在反例风险。

**经典反例**：
> "天鹅₁是白的，天鹅₂是白的，…，没有黑天鹅，所以所有天鹅都是白的。"
> ——后来发现澳洲黑天鹅，结论被推翻。

#### 类型二：科学归纳推理

**核心改进**：结合了**因果联系分析**，结论可靠性显著高于简单枚举。

**逻辑形式**：
```
S₁具有属性P，原因是C
S₂具有属性P，原因是C
…
所有S都具有属性P，原因是C
∴ 所有S都具有属性P（因果机制已知，结论可靠性高）
```

**示例**：
> "金属₁受热体积膨胀，原子间距增大；金属₂受热体积膨胀，原子间距增大；…所有金属受热原子间距增大，所以所有金属受热体积膨胀。"
> ——结论可靠性高，因为揭示了因果机制。

**语文教学应用**：
- "归纳文章中心思想"：不能只举几个段落，要结合全文因果分析
- "总结段落大意"：科学归纳法要求分析段落间的逻辑关联


### 6.1 求同法（Agreement）

**核心思想**：**异中求同**——在被研究现象出现的多个不同场合中，如果只有一个共同的情况，那么这个共同情况极有可能是该现象的原因。

**逻辑形式**：
```
场合1：有先行情况A、B、C，有被研究现象a
场合2：有先行情况A、D、E，有被研究现象a
场合3：有先行情况A、F、G，有被研究现象a
∴ A是a的原因
```

**示例**：
> 某班学生在一次突发性食物中毒事件中，多人出现呕吐腹泻症状。
> 调查发现：这些学生中午都吃了学校食堂的凉拌菜，
> 而其他没吃凉拌菜的学生则安然无恙。
> ∴ 凉拌菜是导致中毒的共同原因。（求同法）

**语文教学应用**：
- 分析多篇同主题文章的共性：如鲁迅多篇杂文都体现"批判国民性"→ 归纳鲁迅创作核心主题
- 作文指导：观察多个成功案例的共同点（如"坚持""善于思考"），归纳成功规律

**局限**：可能存在多个共同情况，需结合其他方法验证。


### 6.3 求同求异并用法（Agreement and Difference Combined）

**核心思想**：**求同法 + 求异法综合运用**。

**三步流程**：
1. **正面求同**：在被研究现象出现的若干正面场合中，找出唯一共同的情况
2. **反面求同**：在被研究现象不出现的若干反面场合中，找出唯一共同的情况（即没有那个共同情况）
3. **正反对比**：对比正反两方面的结果，最终确定因果联系

**逻辑形式**：
```
正面场合群（有现象a）：有A、B、C；有A、D、E；有A、F、G……
反面场合群（无现象a）：无A、有B、C；无A、有D、E；无A、有F、G……
∴ A是a的原因
```

**示例**：
> 研究吸烟与肺癌的关系：
> - 正面场合（肺癌患者）：绝大多数都有长期吸烟史（共同情况A）
> - 反面场合（健康人群）：绝大多数都没有吸烟史（缺乏共同情况A）
> ∴ 吸烟是导致肺癌的重要原因。（求同求异并用法）

**语文教学应用**：
- 评价历史人物/事件：分析"改革成功"案例（如商鞅变法）和"改革失败"案例（如王安石变法）
  → 找出成功案例共有的条件（如君主强力支持、改革措施系统）和失败案例共有的条件 → 归纳改革成功的关键因素


### 6.5 剩余法（Residues）

**核心思想**：**从余果推余因**——已知某一复杂现象是由另一复杂原因引起的，如果把其中已确认因果关系的部分减去，那么剩余部分的现象与剩余部分的原因之间，就存在因果联系。

**逻辑形式**：
```
已知：复合原因A、B、C、D是复合现象a、b、c、d的原因
又知：B是b的原因，C是c的原因，D是d的原因
∴ A是a的原因
```

**经典案例（居里夫人发现镭）**：
> 居里夫人发现从沥青铀矿中提炼出的铀，其放射性比纯铀的放射性强得多。
> 她推断：既然矿石的放射性是复合现象，而纯铀的放射性是已知部分，
> 那么剩余的更强的放射性，一定来自于矿石中一种未知的、放射性更强的元素。
> 她最终通过化学方法分离出了这种新元素——镭。
> ∴ 剩余部分的放射性来自未知元素（剩余法）。

**语文教学应用**：
- 语文考试中"原因分析"类题目：分析"某朝代衰落"的原因，
  已知有"政治腐败""土地兼并""外族入侵"等原因，
  但衰落速度超乎寻常 → 分析是否还存在未被注意的"内部农民起义"或"重大自然灾害"等剩余原因


## 第三节 类比推理

### 核心定义

**类比推理**：根据两个或两类对象在某些属性上相同，推出它们在**其他属性上也相同**的推理。

**逻辑形式**：
```
对象A具有属性a、b、c、d
对象B具有属性a、b、c
∴ 对象B也具有属性d
```

### 或然性分析

类比推理的结论**或然性很高**，需要更多证据验证。

**可靠性提升条件**：
1. 相同属性越多，结论可靠性越高
2. 相同属性与推出属性的相关性越强，结论可靠性越高
3. 没有反例

### 经典示例

> **火星有大气层、有水、有生命**
> **地球有大气层、有水**
> ∴ **地球也有生命**（实际正确，但类比推理本身不保证结论必真）

> **类比推理的经典反例**：
> "火星有大气层、有水、有昼夜交替，地球也有这些属性，且地球有生命，所以火星也有生命。"
> ——这个结论在当代已被部分验证（发现火星曾有水），但类比推理本身无法保证结论必真。


## 核心金句

1. **"归纳推理是从个别到一般的推理，结论超出前提范围，具有或然性。"**
2. **"穆勒五法是归纳推理的核心工具——求同、求异、并用、共变、剩余，五种方法各有适用场景。"**
3. **"完全归纳结论必然真，不完全归纳结论或然——科学归纳法通过揭示因果机制提升可靠性。"**
4. **"类比推理是根据两个对象某些属性相同推出其他属性也相同的推理，结论或然性最高。"**
5. **"语文教学中的'仿写'训练，本质就是类比推理的应用。"**
