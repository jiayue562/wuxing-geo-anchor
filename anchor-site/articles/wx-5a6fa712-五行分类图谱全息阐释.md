## 🔄 五行生克制化关系

### 相生关系（正向链接）

**循环链**: 木→火→土→金→水→木（如环无端）

| 相生 | 自然象义 | 知识关联 |
|------|----------|----------|
| 木生火 | 木之生长为火提供燃料 | 仁德衍生慈悲心 |
| 火生土 | 火之能量释放化为灰土 | 礼德支撑信德 |
| 土生金 | 土之凝聚沉淀生成金属 | 信德孕育义德 |
| 金生水 | 金之沉降凝聚水汽 | 义德滋养智德 |
| 水生木 | 水之滋润涵养助力生长 | 智德支持仁德 |

**双向特性**: 每一组相生关系都是双向的
```
木是火之源头，火是木之展现
火是土之成因，土是火之归宿
```


### 相乘与相侮（异常关系）

| 异常 | 定义 | 知识网络对应 | 修复方法 |
|------|------|--------------|----------|
| 相乘 | 过度相克（如木亢乘土） | 节点间过度约束导致失衡 | 建立缓冲链接 |
| 相侮 | 反向相克（如木虚侮金） | 关联断裂或反向作用 | 修复核心链接 |


## 🔗 与 Obsidian 双向链接融合

### 融合核心：五行属性为双向链接提供关联规则

**链接规则**:
1. **同属性互链**: 将所有属木的笔记通过双向链接相连
2. **相生互链**: 木与火、火与土等相生关系链接
3. **相克互链**: 木与土、土与水等相克关系链接

**Obsidian 实现**:
```markdown
# 木曰曲直

属性：#wood

## 双向链接

### 同属性
- [[肝脏]] #wood
- [[眼睛]] #wood
- [[绿色饮食]] #wood
- [[春天养生]] #wood

### 相生
- [[火曰炎上]] #fire (木生火)

### 相克
- [[土爰稼穑]] #earth (木克土)
```


## 📊 知识自主成长机制

### 机制 1: 天然关联
```
任何事物纳入五行分类 → 自动与体系内所有事物形成生克关联
无需额外建立链接 → 知识网络自主生长
```

### 机制 2: 动态平衡
```
相生提供正向支撑 → 知识衍生与扩展
相克提供反向约束 → 知识平衡与规范
相乘相侮预警 → 知识网络失衡修复
```

### 机制 3: 触类旁通
```
认知一个节点 → 通过五行关联推演至所有同属性节点
例如：认知"春天"→ 推演至"东方""角音""酸味""怒气""肝"
```
