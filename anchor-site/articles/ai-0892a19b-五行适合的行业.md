## 二、火行人适合的行业

### 1. 火的行业：广告传媒

- **行业特点**：火主热情、表达、传播、传媒
- **合作伙伴**：宜选择东方的合作伙伴（东方为木，木生火）；其次也可选择南方的合作伙伴（南方属火互相帮助）
- **风险控制**：火类行业没有不宜属相的风险年，与火为南方所属有关，传播能力强，该行业抗风险能力也强

### 2. 火行人适合的职业

| 职业 | 匹配原因 |
|------|---------|
| **销售业务** | 火主热情、表达，适合销售业务 |
| **市场人员** | 火主传播、推广，适合市场人员 |
| **公关/媒介** | 火主沟通、连接，适合公关媒介 |
| **广告/会展** | 火主创意、展示，适合广告会展 |
| **汽车销售与服务** | 火主动力、速度，适合汽车销售 |
| **化工** | 火主化学反应、能量，适合化工 |
| **影视/媒体/出版/印刷** | 火主传播、表达，适合影视媒体 |
| **烹饪/料理/食品研发** | 火主热能、美味，适合烹饪料理 |
| **保健/美容/美发/健身** | 火主美丽、健康，适合保健美容 |


## 四、金行人适合的行业

### 1. 金的行业：IT/互联网、金融投资、交通运输、制造行业

- **行业特点**：金主收敛、规则、秩序、制造
- **合作伙伴**：宜选择中央或本地的合作伙伴（中央或本地为土，土生金）；其次也可选择西方的合作伙伴（西方属金互相帮助）
- **风险控制**：金类行业没有不宜属相的风险年，与金为西方所属有关，规则秩序强，该行业抗风险能力也强

### 2. 金行人适合的职业

| 职业 | 匹配原因 |
|------|---------|
| **高级管理CEO** | 金主收敛、规则，适合高级管理 |
| **互联网产品/运营管理** | 金主规则、秩序，适合产品运营 |
| **IT质量管理/测试/配置管理** | 金主质量、标准，适合质量管理 |
| **银行** | 金主金融、收敛，适合银行 |
| **证券/期货/投资管理** | 金主投资、收敛，适合证券期货 |
| **交通运输服务** | 金主秩序、运输，适合交通运输 |
| **生产管理/运营** | 金主制造、管理，适合生产管理 |
| **电子/电器/半导体/仪器仪表** | 金主电子、精密，适合电子电器 |
| **汽车制造** | 金主制造、精密，适合汽车制造 |
| **机械设计/制造/维修** | 金主机械、制造，适合机械制造 |
| **技工/操作工** | 金主操作、精密，适合技工操作 |
