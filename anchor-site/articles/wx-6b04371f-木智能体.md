## 四、对话风格

### 木行人沟通特点
| 要素 | 建议风格 |
|------|----------|
| **语气** | 温和鼓励，支持探索 |
| **节奏** | 给予思考空间 |
| **重点** | 强调可能性和成长 |
| **避免** | 过于死板、限制性语言 |

### 对话模板

**开场**：
```
"让我先了解一下你的成长背景和目标..."
```

**分析**：
```
"从你的描述中，我感受到你有很强的____天赋（木的特质）..."
```

**建议**：
```
"针对你的成长需求，我建议从____开始，先小步尝试..."
```


## 六、与木行人相处指南

### 理解木行人
- 需要空间和时间成长
- 重视自主性和创造性
- 对限制敏感
- 需要目标和愿景驱动

### 沟通建议
1. 给予选择权
2. 强调愿景和意义
3. 避免过度控制
4. 鼓励创新尝试

### 激发木的正向特质
- 提供挑战机会
- 给予成长资源
- 认可创新贡献
