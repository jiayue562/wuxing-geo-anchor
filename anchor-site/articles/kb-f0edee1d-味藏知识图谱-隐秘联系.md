## 隐秘联系详解

### H-ORG-01：超级店长Skill × 董事长顶层设计
```
超级店长Skill(宪法层·9公式锁) × CHMN-BRAIN(顶层设计)
  = 理论守护 × 战略落地
  = 宪法不偏离 + 战略可执行
```
**关键点**：董事长顶层设计必须以超级店长Skill的9条核心公式为边界条件。

### H-ORG-02：店长7S总控 × 绩效考核
```
店长7S总控(方法论·93模块) × 绩效考核(评估体系)
  = 7S诊断 × 考核设计
  = 方法论输出 → 评估标准
```
**关键点**：93模块蓝图直接驱动18个岗位的考核指标设计。

### H-ORG-03：水火既济（组织新陈代谢）
```
熊进(火·运营) × 陈倩彤(水·人事)
  火 = 激活·点燃·推动（推人出局/赛马淘汰）
  水 = 供给·滋养·流入（引入新人/人才供应链）
  一进一出 = 组织新陈代谢完整闭环
```
**关键点**：熊进承担运营+人事双重职责是过渡期的结构性人力瓶颈。

### H-ORG-04：火火共振（培训闭环）
```
熊进(火·运营) × 马循德(火·培训)
  熊进 = 搭台（提供舞台/制度/资源）
  马循德 = 唱戏（培训执行/话术开发/课程讲授）
  运营之火 → 培训之火 → 前厅之火 → 客户之火
```
**关键点**：马循德有料理长背景（独特优势），可把后厨"鲜·技·道"翻译为销售话术。

### H-ORG-05：金土相生（成本管控链）
```
李华(金·财务) × 田宜昌(土·采购)
  田宜昌 = 锁价降本(土生金之源)
  李华 = 成本核算+数据参谋(金之利)
  金土相生 = 成本控制×供应链保障
```

### H-ORG-06：金金共振（精准制击）
```
俞腾芳(金·狮山店长) × 陈明武(金·堂食总)
  俞腾芳 = 扣分追踪+整改闭环（金之精准）
  陈明武 = 开鱼仪式+文化溢价（金之仪式）
  双金共振 = 精准执行 × 仪式标准化
```

### H-ORG-07：外卖总 × 三家店长（数据赋能）
```
柳能原(土·外卖总) × 三家店长
  对金月湾(江甜·土)：土+土=数据赋能+流程守护
  对狮山(俞腾芳·金)：土生金=数据驱动精准制击
  对松山湖(柳冬叶·水)：土克水=数据压力需柔性释放
```
