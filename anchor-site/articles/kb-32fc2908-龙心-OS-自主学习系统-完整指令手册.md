## 🏗️ 第一层：持久记忆系统

### 指令一：写入记忆

```
帮我把以下内容写入长期记忆：[你想记住的内容，比如：我的项目是 XX，我的偏好是 XX，我学到了 XX]
写入路径：MEMORY.md
```

**用途**: 把重要信息、偏好、结论永久保存，下次对话直接调用。

**实现方式**:
- 使用 `edit` 工具追加到 `MEMORY.md`
- 自动分类到对应板块
- 添加时间戳和上下文

**示例**:
```
帮我把以下内容写入长期记忆：悟空偏好直接、清晰的沟通风格，不喜欢客套话
```


### 指令三：记忆蒸馏

```
请读取我的 MEMORY.md，找出其中重复、过时、冗余的内容，压缩整理成更精炼的版本，保留所有核心信息，去掉废话。
```

**用途**: 记忆积累多了之后定期清理，保持精炼可用。**建议每月执行一次。**

**实现方式**:
1. 读取完整 MEMORY.md
2. 识别重复内容
3. 标记过时信息
4. 压缩整理
5. 生成新版本

**执行频率**: 每月一次（可配置 cron 自动执行）


### 指令五：深度研究（多角度）

```
对 [话题] 进行深度研究，从以下维度分析：
1. 现状与趋势
2. 核心数据与案例
3. 对我的直接影响
4. 行动建议
```

**用途**: 需要全面了解一个领域时使用，比指令四更系统深入。

**执行流程**:
1. 多引擎搜索获取广度
2. 深度阅读关键页面
3. 交叉验证信息
4. 结构化输出

**输出格式**:
```
# [话题] 深度研究报告

## 1. 现状与趋势
[当前状态 + 发展方向]

## 2. 核心数据与案例
[关键数据 + 典型案例]

## 3. 对我的直接影响
[与龙心 OS/悟空的关联]

## 4. 行动建议
[具体、可执行的步骤]
```


### 指令七：批量读取知识库

```
读取我的知识库中，关于 [主题] 的所有笔记（路径：[你的知识库路径]），综合整理后输出结构化知识图谱。
输出格式：
1. 总体架构图
2. 核心结论（直接可用）
3. 知识节点关系图
4. 行动建议
```

**用途**: 批量读取 Obsidian 或其他本地知识库，整合成结构化知识图谱。

**知识库路径**:
- **Obsidian**: `D:\以观其妙书院知识库\观其妙书院`
- **OpcCla**: `C:\Users\jia'yue\.OpcCla 知识库`
- **OpenClaw**: `C:\Users\jia'yue\.openclaw\workspace`

**实现方式**:
1. 扫描指定路径下所有 `.md` 文件
2. 语义搜索与主题相关的内容
3. 提取实体和关系
4. 生成知识图谱 (Mermaid)
5. 综合输出结构化报告


### 指令九：设置定时学习任务

```
帮我设置一个每周一早上 9 点自动运行的任务：读取过去 7 天的工作记忆，整理成本周知识摘要，写入 MEMORY.md 的"周摘要"板块。
```

**用途**: 设置一次，之后每周自动执行。系统在你不用它的时候也在工作。

**实现方式**:
```bash
openclaw cron add --schedule "0 9 * * 1" --command "weekly-knowledge-summary"
```

**任务内容**:
1. 读取 memory/ 目录下过去 7 天的日常笔记
2. 提取关键事件、决策、学习点
3. 生成周摘要（结构化格式）
4. 追加到 MEMORY.md 的"周摘要"板块
5. 可选：同步到 Obsidian 和 IMA

** cron 表达式参考**:
| 频率 | 表达式 | 说明 |
|------|--------|------|
| 每日 | `0 2 * * *` | 每天凌晨 2 点 |
| 每周 | `0 9 * * 1` | 每周一上午 9 点 |
| 每月 | `0 10 1 * *` | 每月 1 日上午 10 点 |


## 💡 三句话理解这个系统

1. **指令 1-3 = 让龙龟记住你，不再每次从零开始**
   - 写入你的偏好、项目、决策
   - 随时调取验证
   - 定期清理保持精炼

2. **指令 4-7 = 让龙龟帮你获取和整理知识**
   - 实时搜索最新信息
   - 深度研究复杂话题
   - 读取各种格式文件
   - 整合知识库内容

3. **指令 8-9 = 让系统自动进化，不需要你操心**
   - 每次对话自动沉淀精华
   - 每周自动整理摘要
   - 系统在你不用它时也在工作


## 🎯 快速开始

### 第一次使用（5 分钟）

1. **写入基本信息** (指令一)
   ```
   帮我把以下内容写入长期记忆：
   - 我的名字：悟空
   - 我的项目：龙心 OS
   - 我的偏好：直接、清晰、有深度的沟通
   ```

2. **验证记忆** (指令二)
   ```
   请读取我的长期记忆，告诉我关于"项目"你记住了什么？
   ```

3. **设置自动化** (指令九)
   ```
   帮我设置每周一早上 9 点的周知识摘要任务
   ```

### 日常使用流程

```
早上 → 指令四/五：获取最新信息
       ↓
工作中 → 指令六/七：读取文件/知识库
       ↓
对话结束 → 指令八：知识沉淀
       ↓
每周五 → 指令三：记忆蒸馏（每月一次）
```
