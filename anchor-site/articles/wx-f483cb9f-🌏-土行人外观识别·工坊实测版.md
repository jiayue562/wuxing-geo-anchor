## 二、体型特征（身界·物象层）

### 2.1 整体体型

| 特征维度 | 描述 | 补充说明 |
|---------|------|---------|
| **体行（整体）** | 敦实，圆厚，圆脸；厚：敦实，肌肉紧致 | 核心：厚重感，不是松垮而是紧致 |
| **头行** | 圆脸，头大，肥，颊宽 | 脸宽大是土行最明显特征 |
| **脸行** | 宽脸（圆脸） | 宽脸+善良表情=土行识别要点 |
| **面色** | 黄，发黑，发黄，发暗 | 土对应黄色；有时偏暗 |
| **肤色** | 黑，发黑，发黄，发暗 | 土行肤色较深 |

### 2.2 五官细节

| 部位 | 特征 | 工坊补充数据 |
|------|------|------------|
| **鼻子** | 大，厚实，挺，鼻头大，肉厚 | 鼻头大肉厚是土行特征 |
| **眼睛** | 双眼皮，明亮，清澈，干净，黑白分明 | 眼睛明亮清澈是土行气质体现 |
| **眉毛** | 浓，浓密 | 眉毛浓密 |
| **耳朵** | 大，耳垂大而厚，硬 | 大耳垂是土行重要特征 |
| **肩背** | 厚 | 肩厚背厚，承载感强 |
| **腹股** | 敦实，厚宽 | 腹部敦实有力 |
| **手足** | 厚，大，结实，肉多，方 | 手大厚实方正 |

### 2.3 体型综合数据

```

肤色：发黑，发黄，发暗

```

### 2.4 识别要诀

> **土行三字经：厚·圆·稳**
> - 厚：体型厚重，肌肉紧致
> - 圆：脸型宽圆，头大
> - 稳：气质稳重，动作踏实


## 四、着装风格（身界·物象层）

| 维度 | 工坊实测数据 |
|------|------------|
| **整体风格** | 中性，休闲，宽松 |
| **特征** | 个性，与众不同，特别 |
| **禁忌** | 不喜华丽，不张扬，不喜欢装饰物 |
| **色彩** | 冷色居多 |

> **洞察**：土行人的着装有趣对立：表面上"不张扬"，却喜欢"个性独特"。这体现了土行人"含而不露"的内在：不追求表面光鲜，但有自己独特的审美。
