### 2.2 Kaizen×PDCA循环

```
Kaizen × PDCA循环 同构图谱
│
├── Kaizen四步法
│   ├── Find（发现问题）
│   │   └── → PDCA Plan（计划）
│   │
│   ├── Analyze（分析原因）
│   │   └── → PDCA Plan（分析）
│   │
│   ├── Propose（提出改进）
│   │   └── → PDCA Do（执行）
│   │
│   └── Test+Standardize（试点→推广）
│       └── → PDCA Check+Act（检查→处理）
│
└── 核心同构
    ├── 全员参与 ↔ PDCA全员循环
    ├── 点滴优化 ↔ 小步迭代
    └── 持续迭代 ↔ 闭环闭合
```
