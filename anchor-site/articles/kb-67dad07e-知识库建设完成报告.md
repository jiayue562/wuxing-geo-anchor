## 🔧 三大核心操作

### Ingest（入库）
```
新资料 → 列方案 → 确认 → 分发 → 加工 → 清理
```

### Query（查询）
```
提问 → 搜索 → 综合回答 → 发现关联 → 建议补链接
```

### Lint（健康检查）
```
孤立页面 / 过时内容 / 缺失引用 / Notes 积压 → 生成报告
```
