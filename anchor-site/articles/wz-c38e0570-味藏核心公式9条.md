## 人机协同核心公式

```
人机分工 = AI(数据/过滤/提醒/模板/预警/流转) 
           + 人(判断/执行/审批/感知/决策)

5%判断域 = 真正需要人类判断的核心只有5%
AI替代域 = 95%人类也能做的信息处理工作
```

**核心金句**：*"AI做95%的信息处理，人做5%的判断决策——这不是替代，是解放"*


## 9条核心公式（F1-F9）

### F1：7S企业生命体
```
7S = SV(DNA) × S(大脑) × St(骨骼) × Sy(神经) × Sty(气质) × Sf(血液) × Sk(肌肉)
```

### F2：超级店长
```
超级店长 = 店长 × AI = 超级店长
```

### F3：店长本质
```
店长本质 = 掌握人员 + 抓经营目标 + 有余力抓重要客户
```

### F4：成功公式
```
成功 = 性格(N4/N5) × 目标(N1) × 能力(N6) × 努力(N3/N2) × 修心(N7)
```
- N1=目标，N2/N3=努力，N4/N5=性格，N6=能力，N7=修心

### F5：营业额
```
营业额 = 堂食(仪式即文化) + 酒水(氛围即开心) + 外卖(信息即增长)
```

### F6：三昧真火
```
三昧真火 = 产品结构 × 会员复购 × 文化匠心
```

### F7：紧箍咒
```
紧箍咒 = 产品稳定 × 上菜快 × 服务及时
```

### F8：质价比
```
质价比 = 远超300元预期的价值
```

### F9：匠心
```
匠心 = 对过程的坚守 = 质价比 = 有序经营
```
