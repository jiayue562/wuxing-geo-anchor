## 🔄 五行生克关系

### 相生循环（生成关系）

```
木 → 火 → 土 → 金 → 水 → 木（循环）
│    │    │    │    │
│    │    │    │    └─ 水生木（滋养生长）
│    │    │    └────── 金生水（凝结成液）
│    │    └─────────── 土生金（矿藏于土）
│    └──────────────── 火生土（灰烬成土）
└───────────────────── 木生火（燃烧生火）
```

**相生含义**：
- **水生木**：水滋养树木生长
- **木生火**：木材燃烧产生火
- **火生土**：火烧尽后成灰土
- **土生金**：金属矿藏于土中
- **金生水**：金属表面凝结水珠（或金属熔化成液态）

### 相克循环（制约关系）

```
木 → 土 → 水 → 火 → 金 → 木（循环）
│    │    │    │    │
│    │    │    │    └─ 金克木（金属砍伐树木）
│    │    │    └────── 火克金（烈火熔化金属）
│    │    └─────────── 水克火（水能灭火）
│    └──────────────── 土克水（土能挡水）
└───────────────────── 木克土（树根穿透土壤）
```

**相克含义**：
- **金克木**：金属工具可以砍伐树木
- **火克金**：烈火可以熔化金属
- **水克火**：水可以灭火
- **土克水**：土可以阻挡水流
- **木克土**：树根可以穿透土壤

### 生克平衡原则

**核心智慧**：
> "生中有克，克中有生，生克制化，动态平衡。"

**过度相生**：
- 母旺子衰：生者过强，被生者反而受损
- 例：水太多，木反而烂根

**过度相克**：
- 克者过强：被克者受损
- 例：金太强，木被过度砍伐

**理想状态**：
- 生克有度，动态平衡
- 生而不溺，克而不伤

- 观察自己的气质、行为模式、偏好

**步骤 2：平衡分析**
