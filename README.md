## 最后更新: 2026-05
### 核心定义
「一心三界五行九层」是悟空独创的五行人格心理学完整理论框架，以「一心」（纯粹觉知/大圆满见地）为根本，通过「三界」（法身/报身/化身，信息体/能量体/物质体）的结构分层，展现「五行」（木火土金水）的动态流转，并以「九层」（健康1-3级/一般4-6级/不健康7-9级）量化人格发展状态，构成完整的生命理解与转化系统。
### 一、总纲逻辑
#### 四维一体结构
```
一心（纯粹觉知·大圆满见地）
    ↓ 显化
三界（法身/报身/化身 = 信息体/能量体/物质体）
    ↓ 流转
五行（木火土金水 = 阴阳两面·生克制化）
    ↓ 量化
九层（健康1-3·一般4-6·不健康7-9）
```
#### 核心关系
| 维度 | 内涵 | 功能 |
|------|------|---------|
| **一心** | 纯粹觉知，超越阴阳五行 | 根本·终极目标·觉性 |
| **三界** | 生命能量的三层显化 | 诊断场域·调适空间 |
| **五行** | 能量动态流转模式 | 分类·识人·转化 |
| **九层** | 发展层级量化光谱 | 评估·定位·引导 |
### 二、一心——纯粹觉知
一心是超越三界五行的纯粹觉知本身。不是任何具体的感受、思想或行为，而是能够觉知这一切的「觉知本身」。
**大圆满见地**：
- **本来清净**（空性）：一切法无任何本质，超越轮回、因果、缘起
- **本自圆满**（光明）：具足一切光明潜能，不增不减
- **不二**（噶达陇竹尼美）：空性与光明本是一体
### 三、三界——能量的三层结构
| 三界 | 传统名称 | 现代对应 | 功能 |
|------|---------|---------|------|
| **灵界** | 法身 | 信息体 | 价值观·使命·信仰·根本动力 |
| **心界** | 报身 | 能量体 | 情绪·思维·心理模式·感受 |
| **身界** | 化身 | 物质体 | 行为·表情·体态·物理表现 |
**三界协同机制**：
健康状态（马车比喻）：
```
灵界（仁德使命）→ 驱动 → 心界（积极情绪）→ 驱动 → 身界（健康行为）
三界方向一致，能量流畅，走运。
```
失衡状态：
```
灵界迷失（阴面价值观）→ 心界紊乱（负面情绪）→ 身界失常（问题行为）
三界方向不一致，能量阻塞，不走运。
```

- **不怨**：放弃怨恨，断掉阴面能量的供给
- **转化**：用拔阴取阳口诀，主动切换到阳面模式
- **践行**：在身界实际行动，完成三界转化闭环

## 最后更新: 2026-05
### 核心定义
五行识人三步法是悟空独创的「观象→取象→运象」象思维实践流程，将识人用人从"技巧"升华为"智慧"。
### 第一步：观象——以开放觉知感知物象
**定义**：保持觉知，不带预设地观察对象在"身界"呈现的**物象**。
**观象要点（身界·物质层面）**：
体型外貌（口诀：**木瘦金方水主肥，土形敦厚背如龟，上尖下阔名为火**）：
| 五行 | 体型特征 | 面部特征 |
|------|---------|---------|
| 木 | 修长清瘦·挺拔 | 面色青白透红·眉目清秀·眼睛有神 |
| 火 | 上尖下阔·锐隆壮实 | 面红润或偏深·眉浓·眼大有神·嘴大 |
| 土 | 敦实圆厚·圆润 | 面色偏黄·眼睛圆大·嘴唇厚 |
| 金 | 匀称方正·清爽 | 面色偏白·颧骨突出·轮廓清晰 |
| 水 | 软圆柔润·偏肥 | 面色偏黑或暗·眼睛细长·嘴唇丰润 |
**动作节奏**：
- 木：能快能慢（随关系亲疏调节节奏）
- 火："嚯"声音标志·动作幅度大·节奏快·爱出声
- 土：走路快（最重要发现！）·说话稳
- 金：动作精准·先容忍→试探→爆发三段式
- 水：小动作多（捋发/扶镜/摸鼻）·动作细腻
### 第二步：取象——从物象体悟意象
**定义**：通过"物象"，体悟其背后的"意象"，推断内在动机、情绪模式和思维偏好（心界）。
**取象工具：主导情绪模式识别**
| 情绪 | 主导五行 | 触发情境 |
|------|---------|---------|
| **怒**（顶撞、抗上、愤懑） | 木 | 被否定主张、被控制、不公正 |
| **恨/喜**（爱出风头、急躁） | 火 | 被忽视、未被认可、争胜心被挑战 |
| **怨**（抱怨、疑虑、迟缓） | 土 | 付出未被回报、变化太快、不稳定 |
| **恼**（批评、刻薄、恼怒） | 金 | 规则被破坏、他人不达标、被评价 |
| **烦/恐**（退缩、疑神疑鬼） | 水 | 被暴露、被要求行动、不确定性 |
### 第三步：运象——从意象触及原象
**定义**：通过"意象"，进一步触及"原象"，理解对象的核心价值观、使命感和人生哲学（灵界）。
**运象工具：核心价值观识别**
| 五行 | 核心追求 | 人生使命 |
|------|---------|---------|
| 木 | 成长·创新·突破 | 带来生机与变革 |
| 火 | 连接·照亮·荣耀 | 传播光明与温暖 |
| 土 | 稳定·和谐·滋养 | 构建平台承载万物 |
| 金 | 秩序·公正·成就 | 建立标准捍卫价值 |
| 水 | 智慧·融合·流动 | 洞察本质维系平衡 |
