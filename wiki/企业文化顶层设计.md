# 企业文化顶层设计

## 核心定义

企业文化顶层设计是一套基于中国本土哲学根基（天文考古→文明论→心文化）的企业文化构建体系，突破西方"理性经济人"假设，以组织三体论（信息体+能量体+物质体）为框架，通过差序格局+礼法合治实现文化落地。

## 五大核心概念

### 1. 组织三体论
- **信息体**：承载使命愿景价值观文化观 → 龙爪上行信号
- **能量体**：驱动团队协作氛围机制 → 断语汇聚
- **物质体**：落地实体组织架构流程制度 → 行动日历
- **突破点**：取代"理性经济人"→强调"创新属性"与"生命特质"

### 2. 心文化
- 易经洗心+中医养心+儒家正心+道家静心+禅宗明心
- 根源：天文观测→规律可信→诚信→内化→从外在约束转向内在驱动
- 映射：[[SOUL.md]]心文化信仰体系六家根基

### 3. 差序格局
- 以己为中心的水波纹社会关系（费孝通）
- 核心圈(礼治)+外围圈(法治)→礼法合治
- 企业适配：企业家协作的天然结构
- 映射：龙爪S1/S2/S3层级=差序格局编码

### 4. 五行企业家精神
- 创新(木)→冒险(火)→诚信(土)→敬业(金)→应变(水)
- 生克动态平衡：木生火(创新驱动变革)/火克金(防规则僵化)/金生水(规则促适应)/水生木(适应催创新)
- 映射：[[五行人格心理学]]五行诊断→岗位适配

### 5. 礼法合治
- 礼主内(主动/习惯/心安)+法主外(契约/清算/异议)
- 中国企业的第三条道路：既非纯集体主义也非纯个人主义
- 映射：龙心OS P0规则(法)+心文化(礼)→双驱动

## 天文考古→企业文化论证链

```
天文观测(陶寺/贾湖/西水坡)
  → 三大宇宙观(天人合一/天圆地方/中轴对称)
    → 天文思想→历法+礼仪+政治制度
      → 文明三本(道德+知识+礼仪)
        → 心文化+组织三体论+礼法合治
          → 企业文化
```

## 行为经济学双锚模型

- **锚1**：损失厌恶(2.5倍痛苦感) → 微观干预(容错/风险共担/福利阶梯)
- **锚2**：锚定效应(初始信息依赖) → 文化锚定(理念体系/行为规范)
- **协同**：选择架构(Choice Architecture)
- **发现**：礼治≈最古老的行为经济学工具
