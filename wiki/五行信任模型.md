# 五行信任模型

以中国传统文化"木火土金水"五行与"仁义礼智信"五常为哲学根基，融合五大西方经典信任理论（成交公式、麦肯锡信任公式、双维信任模型、信任阶梯模型、信任五波模型），构建的动态信任生态系统。

## 核心公式

```
五行信任模型 = 东方哲学根基（五行+五常）× 西方方法论（五大信任理论）
```

**三大核心跃迁**：从拆解到融合 → 从四维到五维（信成为中枢）→ 从线性递进到相生循环

## 五维体系

| 五行 | 五常 | 信任维度 | 麦肯锡变量 | 双维通道 | 信任阶梯 |
|------|------|---------|-----------|---------|---------|
| 木 | 仁 | 立场共鸣 | 自我导向↓ | 情感信任 | 打破怀疑 |
| 火 | 礼 | 标准可视 | 可信度↑ | 认知信任 | 建立初步信任 |
| 土 | 信 | 持续兑现 | 全公式底座 | 双通道稳定 | 贯穿全阶梯 |
| 金 | 义 | 顾虑前置 | 亲密度↑+可靠度↑ | 情感信任为主 | 建立深度信任 |
| 水 | 智 | 结果实证 | 可靠度↑(闭环) | 认知信任为主 | 验证→拥护 |

## 相生循环

```
木生火(仁生礼) → 火生土(礼生信) → 土生金(信生义) → 金生水(义生智) → 水生木(智生仁)
```

每完成一次循环，信任水平螺旋上升一个层级。

## VDT信任指数

```
VDT = [(认知信任×0.5 + 情感信任×0.5)] / 自我导向
认知信任 = 火(标准可视)×0.5 + 水(结果实证)×0.5
情感信任 = 木(立场共鸣)×0.4 + 金(顾虑前置)×0.6
土·信 = 前四维多次测量的方差稳定性
```

## 验证案例

- **胖东来**：擀面皮事件(2024.06)→寿司事件→猪头肉价签→粉条铝残留，每次处理节奏一致，复购率92%，坪效行业2.8倍
- **张雪机车**：820RR事件(2026.04-05)，五天内密集动作完成完整相生循环，从初始信任跃迁至品牌信任
